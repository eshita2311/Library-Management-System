# Library-Management-System
Java + Swing + MySQL Library Management System
## Features
- Login system
- Add / Remove books
- Issue & Return books
- View available and issued books
- MySQL database integration

## Tech Stack
- Java (Swing)
- MySQL
- JDBC
- NetBeans

## How to Run
1. Import project in NetBeans
2. Configure MySQL database
3. Run the project
